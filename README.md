# Material-Testing-under-LEO-Conditions-
Emily BWM is part of team STELLAR, research team that identified PI-B-PDMS and CFR as potential materials for use as protective coatings for LEO conditions. The objective of this repository is to verify and validate the steps of STELLAR during pre flight and post flight of the SPIRIT-02 Mission, which will take place in fall 2027.
